# شركة مسفر محمد العرجاني للمحاماة والاستشارات القانونية

نظام إدارة شامل للمحاماة يتضمن موقع عام، لوحة تحكم للعملاء، ولوحة تحكم إدارية.

## المتطلبات

- PHP >= 8.2
- Composer
- Node.js & NPM
- MySQL أو MariaDB

## التثبيت

### 1. تثبيت المتطلبات

```bash
composer install
npm install
```

### 2. إعداد البيئة

انسخ ملف `.env.example` إلى `.env`:

```bash
cp .env.example .env
```

قم بتعديل ملف `.env` وأضف معلومات قاعدة البيانات:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=law_firm
DB_USERNAME=root
DB_PASSWORD=your_password
```

### 3. توليد مفتاح التطبيق

```bash
php artisan key:generate
```

### 4. إنشاء قاعدة البيانات

```bash
php artisan migrate --seed
```

### 5. بناء الأصول

```bash
npm run build
```

أو للتطوير:

```bash
npm run dev
```

## تشغيل المشروع

```bash
php artisan serve
```

افتح المتصفح على: `http://localhost:8000`
