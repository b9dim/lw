# تعليمات بناء الأصول (Assets)

## المشكلة
التصميم المحدث موجود في ملف `resources/css/app.css` لكنه لم يتم تجميعه بعد في ملفات الإنتاج.

## الحل

### الطريقة 1: استخدام npm (الأفضل)

1. تأكد من تثبيت Node.js من: https://nodejs.org/

2. افتح Terminal/PowerShell في مجلد المشروع

3. قم بتثبيت الحزم (إذا لم تكن مثبتة):
```bash
npm install
```

4. قم ببناء الأصول:
```bash
npm run build
```

أو للتطوير مع إعادة التحميل التلقائي:
```bash
npm run dev
```

### الطريقة 2: استخدام ملف batch

قم بتشغيل الملف:
```
build-assets.bat
```

### بعد البناء

1. قم بتحديث الصفحة في المتصفح (Ctrl+F5 لإجبار التحديث بدون cache)

2. تأكد من أن الخادم يعمل:
```bash
php artisan serve
```

## ملاحظات

- إذا كنت في وضع التطوير، استخدم `npm run dev` بدلاً من `npm run build`
- `npm run dev` سيراقب التغييرات ويقوم بإعادة البناء تلقائياً
- تأكد من مسح cache المتصفح (Ctrl+Shift+Delete)
